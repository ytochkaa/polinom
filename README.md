# polinom
Poly - этот репозиторий содержит реализацию основных методов теории чисел и работы с многочленами над конечными полями, применяемых в криптографии. Класс Poly для работы с многочленами над конечным полем GF(p).
## Описание
- Поддержка операций сложения, вычитания, умножения многочленов.
- Деление многочленов с вычислением частного и остатка.
- Вычисление НОД (GCD) многочленов.
- Проверка неприводимости многочленов.
- Возведение x в степень по модулю многочлена.
- Ввод многочлена вручную из консоли.
